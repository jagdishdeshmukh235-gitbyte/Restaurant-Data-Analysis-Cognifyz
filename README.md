# 📌 Project Overview

This project was completed as part of the **Cognifyz Technologies Data Science Internship**.

The objective is to analyze restaurant data, extract meaningful business insights, visualize customer behavior, engineer useful features, and build machine learning models for restaurant rating prediction.

# 🎯 Objectives

- Explore and preprocess restaurant data
- Perform Exploratory Data Analysis (EDA)
- Analyze restaurant locations
- Study customer preferences

# 🤖 Machine Learning

Built and compared multiple regression models.

| Model | Purpose |
|--------|----------|
| Linear Regression | Baseline Prediction |
| Decision Tree | Non-linear Prediction |
| Random Forest | Ensemble Learning |

The best-performing model was selected based on **R² Score** and prediction error metrics.

# 🚀 How to Run

### Clone Repository

```bash
git clone https://github.com/jagdishdeshmukh235-gitbyte/Restaurant-Data-Analysis-Cognifyz.git
```

### Move to Project Folder

```bash
cd Restaurant-Data-Analysis-Cognifyz
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Launch Jupyter Notebook

```bash
jupyter notebook
```
